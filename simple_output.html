Welcome to HTML Generation Library

This is a simple example demonstrating the HTML generation library. The library allows you to programmatically create HTML documents with proper structure and formatting.

Features Demonstrated

  • Creating HTML document structure
  • Adding CSS styling
  • Creating nested elements
  • Adding paragraphs and headings
  • Using classes and IDs

The library maintains proper HTML structure and automatically handles element nesting and indentation for readable output.
